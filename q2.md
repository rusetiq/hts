### 1035. Tallest Milk Fence

You are given an integer array `heights` where each element represents the height of a vertical fence post.  
Two fence posts can hold up a **stretch of milk fabric** between them. Return the **maximum milk** that can be held between any two fence posts.

---

### Example 1:

**Input:**  
`heights = [1,8,6,2,5,4,8,3,7]`

**Output:**  
`49`

---

### Example 2:

**Input:**  
`heights = [1,1]`

**Output:**  
`1`

---

### Constraints:

- `2 <= heights.length <= 10^5`  
- `0 <= heights[i] <= 10^4`
